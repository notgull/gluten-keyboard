# gluten-keyboard key set, by notgull
# The following is licensed under the MIT and Apache2 licenses.
#
# This is largely inspired by Java AWT's VK_* set, which can be found here:
# https://docs.oracle.com/javase/7/docs/api/java/awt/event/KeyEvent.html
# It also has keys from the following dataset:
# https://github.com/substack/node-keysym/blob/master/data/keysyms.txt
# 
# This should cover pretty much every key on the keyboard, and has the following
# format:
# 
# * The name of the key
# * If the key corresponds to a unicode character, that unicode character's number.
#   (in hexadecimal, if you were wondering).
# 
# Adding to this list is not considered a breaking change for this crate. Modifiying
# or removing any entries from this list is considered a breaking change. For an
# example of parsing this list, see the code in generator/ in the repository root.
